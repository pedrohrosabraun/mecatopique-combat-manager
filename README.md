Combat tracker for the Mecatopique RPG campaign.

📖 Session Information

At the top of the page you can enter:

Session Name
Session Date

These values are automatically saved.

👥 Roster

The roster contains the permanent party members.

For each player:

Name:
Max HP:

Changes are automatically saved.

You can also use:

💾 Save All Data for manual saving.

⚙️ Combat Setup

Start New Combat: Creates all roster members as combatants.
Reset Combat: Removes all combatants and resets the encounter.

Add Mob(s)
Fill:
Field	Description
- Mob Name // Name of the creature
- Max HP	Maximum health of the mob
- VP	Vigor Points of the mob
- Number of Mobs	Number of copies

Example:

Goblin
40 HP
2 VP
5 copies

Creates:
Goblin #1
Goblin #2
Goblin #3
Goblin #4
Goblin #5


🎲 Initiative

Each combatant has:

Roll:	Initiative roll 0-100
Mod	Initiative modifier for iniative
Final:	Roll + Modifier

Combatants are automatically sorted by Final Initiative.

Negative initiatives are allowed.

▶️ Turn Management

Current turn is highlighted.
Acted Early / End Turn: Here is when someone finishes their turn OR uses a Prepared Action (loses his main action/turn)

Situation 1: Before the turn of the mob or player if he uses Prepare Action:
Use the button "Acted Early/End Turn" and it wiil mark the combatant as having already acted, even before his turn.

Situation 2: During the turn, when the turn is finished:
Ends the current on the button "Acted Early/End Turn" turn and advances to the next combatant.

❤️ HP

HP is displayed as:
Current HP / Max HP

Damage
Select a value: 0-100
Click: Apply Damage

Heal
Select a value: 0-100
Click: Apply Heal

🔄 New Round

Resets: "Acted Early/End Turn" for all combatants.

💾 Saving
The tracker uses browser local storage.
Data is automatically saved.
The same device and browser will reopen the tracker exactly where it was left.
